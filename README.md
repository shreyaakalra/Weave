# Weave

**Real connections. Real people.**

Weave is an onboarding-driven social platform that matches people based on **personality, energy, conversation style, and intent** — not just superficial interests.

---

## Features

- Thoughtful onboarding flow (10+ steps)
- Personality-based matching inputs
- Clean, modern UI with smooth animations
- Built using Next.js + Tailwind + Framer Motion
- Modular component structure
- Graph-based matching concept (TigerGraph-inspired)

---

## Onboarding Experience

Weave guides users through a series of meaningful questions:

- Who you are (name, vibe)
- What you enjoy
- How you communicate
- Your social energy
- What you're looking for
- Your availability
- Personal prompts (bio, comfort, boundaries)

This helps create a **deep profile**, not just a surface-level one.

---

## Tech Stack

- **Frontend:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript
- **State Management:** React useState
